# 🚦 EventJam

**EventJam** is an event-aware congestion predictor for Bangalore that shows what events are happening in different areas, predicts their impact on traffic, and suggests the best departure times, risk levels, and transport modes using simulated data.

---

## 💡 Problem

Urban navigation apps do not account for real-world events like:
- Cricket matches 🏏  
- Festivals 🎭  
- Office rush hours 💼  

These events cause sudden congestion, leading to:
- Unexpected delays  
- Poor route decisions  
- Inefficient commutes  

---

## 🚀 Solution

EventJam predicts traffic **before it happens** by analyzing nearby events and their impact on congestion.

It helps users make smarter decisions by providing:
- 🚨 Congestion risk level (High / Medium / Low)  
- ⏱️ Optimal departure time  
- 🚦 Transport recommendations  
- 📍 Event insights near the route  

---

## ⚙️ Features

- 📍 Detects events in key Bangalore areas (MG Road, Chinnaswamy, Koramangala, Whitefield, etc.)  
- 🚨 Real-time-like congestion prediction using simulated data  
- ⏱️ Smart departure time suggestions  
- 🚗 Transport mode recommendations  
- ⚡ Instant, lightweight micro-solution
